Fibonacci Series


Logic:
  • Initialize the first and second number as 0 and 1
  • Print first and second number
  • From the next number, start your loop. So third number will be the sum of the first two numbers

Program:
<?php
$num = 0;
$a = 0;
$b = 1;
echo $a.<br>.$b;
while($num<8)
{
 $c = $a + $b;
 echo"<br>".$c;
 $a = $b;	
 $b = $c;
 $num += 1;
}
?>
