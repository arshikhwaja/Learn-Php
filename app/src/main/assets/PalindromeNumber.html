Palindrome Number


Logic:
  • Take a number
  • Reverse the input number
  • Compare the two number
  • If equal, it means number is palindrome

Program:
<?php				
$num = 12121;
$rev = 0;
$x = $num;
while($num>0)
{
 $rem = $num % 10;
 $rev *= 10 + $rem;	
 $num /= 10;
}
if($x == $rev)
 echo $x. " is a palindrome number ";
else
 echo $x. " is not a palindrome number ";	
?>

Output:

  12121 is a palindrome number
