Even Odd Number


Logic:
  • Take a number
  • Divide it by 2
  • If the remainder is 0, print number is even
  • Otherwise print number is odd

Program:
<?php				
$num = 1629;
if($num%2==0)
{
 echo $num." is an even number";
}
else
{
 echo $num." is an odd number";
}
?>

Output:

   1629 is an odd number
