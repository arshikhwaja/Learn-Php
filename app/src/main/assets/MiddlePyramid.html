Middle Pyramid


Logic:
  • Take three loop, one for counter, second for printing space and third for printing stars
  • Execute the loop till the given condition and increement both the loop and decrement the loop that prints space

Program:
<?php				
for($i=0; $i<=5; $i++)
{
 for($j=5; $j>=$i; $j--)
 {
  echo "  ";
 }
 for($k=1; $k<=$i; $k++)
 {
  echo "* ";
 }
 echo "<br>";
}
?>

Output:

      *
     * *
    * * *
   * * * *
  * * * * *
