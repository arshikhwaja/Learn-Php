Install PHP

To install PHP, we will suggest you to install AMP(Apache,MySQL,PHP) software stack. It is available for all operating system. There are many AMP optios available in the market that are given below:

  • WAMP for Windows
  • Lamp for Linux
  • MAMP for Mac
  • SAMP for Solaris
  • FAMP for FreeBSD
  • XAMPP (Cross, Apache, MySQL, PHP,Perl) for cross-Platform: It includes such other component too such as FIleZilla, OpenSSl, Webalizer, Mercury Mail etc.
