PHP Call By Value

PHP allows you to call function by value and reference both. In case of PHP call by value, actual value is not modified if it is modified inside the function.

Syntax: 

	<?php
	
		function functionName($a) 
		{
			return $a++;

		}
		$a=//value to be assigned
		
	?>

The following example has a function with one argument ($a). When the function is called, the value of a will only be changed inside the function.It will remain unchanged outside the function. File: callbyv.php

	⁢!DOCTYPE>
	<html>
	 <body>
	   <?php
	   function myfunction($a)
           {
           return $a++; 
           echo $a;
           } 
            $a=4;
            myfunction($a);
            echo $a;
	   ?>	
	 </body>
