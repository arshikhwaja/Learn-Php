Leap Year


Logic:
  • Take a year
  • Divide it by 4
  • If the remainder is 0, print year is a leap year
  • Otherwise print year is not a leap year

Program:
<?php				
$year = 2016;
if($year%4==0)
{
 echo $year." is a leap year";
}
else
{
 echo $year." is not a leap year";
}
?>

Output:

   2016 is a leap year
