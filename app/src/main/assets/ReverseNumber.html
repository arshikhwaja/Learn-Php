Reverse Of a Number


Logic:
  • Declare a variable to store reverse number and initialize it with zero
  • Multiply the reverse number with 10, add the remainder which comes after dividing the number by 10

Program:
<?php				
$num = 1234;
$rev = 0;
while($num>1)
{
 $rem = $num % 10;
 $rev *= 10 + $rem;	
 $num /= 10;
}
echo"Reverse of number 1234 is ".$rev;
?>

Output:

  Reverse of number 1234 is 4321
