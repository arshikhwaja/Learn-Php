PHP Array Functions

The array functions allow you to access and manipulate arrays.Simple and multi-dimensional arrays are supported.
    1.array() : Creates an array

    2.array_change_key_case():Changes all keys in an array to lowercase or uppercase

    3.array_chunk():Splits an array into chunks of arrays

    4.array_column():Returns the values from a single column in the input array

    5.array_combine():Creates an array by using the elements from one "keys" array and one "values" array

    6.array_count_values():Counts all the values of an array

    7.array_diff(): Compare arrays, and returns the differences (compare values only)

    8.array_diff_assoc():Compare arrays, and returns the differences (compare keys and values)

    9.array_diff_key():Compare arrays, and returns the differences (compare keys only)

    10.array_multisort():Sorts multiple or multi-dimensional arrays

    11.array_pad():Inserts a specified number of items, with a specified value, to an array

    12.array_pop():Deletes the last element of an array

    13.array_product():Calculates the product of the values in an array

    14.array_push():Inserts one or more elements to the end of an array

    15.array_rand():Returns one or more random keys from an array

    16.array_reduce():Returns an array as a string, using a user-defined function

    17.array_replace():Replaces the values of the first array with the values from following arrays

    18.array_replace_recursive():Replaces the values of the first array with the values from following arrays recursively

    19.array_reverse():Returns an array in the reverse order

    20.array_search():Searches an array for a given value and returns the key

    21.array_shift():Removes the first element from an array, and returns the value of the removed element

    22.array_slice():Returns selected parts of an array

    23.array_splice():Removes and replaces specified elements of an array

    24.array_sum():Returns the sum of the values in an array

    25.array_unique():Removes duplicate values from an array

    26.array_unshift():Adds one or more elements to the beginning of an array

    27.array_values():Returns all the values of an array

    28.array_walk():Applies a user function to every member of an array

    29.array_walk_recursive():Applies a user function recursively to every member of an array

    30.arsort():Sorts an associative array in descending order, according to the value

    31.asort():Sorts an associative array in ascending order, according to the value
