Prime Number


A number which is only divisible by 1 and itself is called prime number.

Logic:
  • Take a number
  • Divide it by 2
  • If the remainder is 0, set flag 0 and break the loop
  • Otherwise print the number is prime

Program:
<?php				
$num = 19;
$flag = 1;
for($i=2;$i<=$num;$i++)
{
 if($i%2==0)
 {
  $flag = 0;
  break;
 }
}
if($flag == 1)
{
 echo $num." is a prime number";
}
else
{
 echo $num." is not a prime number";
}
?>

Output:

   19 is a prime number
