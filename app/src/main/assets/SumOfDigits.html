Sum Of Digits


Logic:
  • Take a number
  • Divide the number by 10
  • Add the remainder to a variable
  • Repeat the process untill remainder is 0

Program:
<?php				
$num = 1234;
$sum = 0;
$rem = 0;
for($i=0;$i<=strlen($num);$i++)
{
 $rem=$num%10;
 $sum+=$rem;	
 $num/=10;
}
echo"Sum of digits 1234 is ".$sum;
?>

Output:

  Sum of digits 1234 is 10
