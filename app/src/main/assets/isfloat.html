PHP is_bool() function

By using this function, we can check that the input value is float or not. This function was introduced in PHP 4.0.

    Syntex :

	<?php	
	bool is_float(mixed $var)
	?>

    Parameter:
     var_name: The variable being evaluated and it is compulsory.

    Returns:
     The is_float() function returns TRUE if the var_name is float, otherwise false.

    Example 1 :

	<?php	
	$a=4;
	echo is_float($a);
	?>

    Output:


               1

    Example 2 :

	<?php	
	$a=5.4;
	is(is_float($a){
	echo $a." is float number!";
        }
	else{
		echo $a." is not float number!";
        }
	?>

    Output:


               5.4 is an float number!
