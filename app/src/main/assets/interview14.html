Answer:

        PHP Constants

        1.In PHP, the constant is an identifier or a name for a simple value. The constant value can't be changed during script execution.

        2.No dollar sign ($) is required before using a constant.

        3.It can't be defined by simple assignment. Only define() function can define it.

        4.It can be redefined or undefined after it gets set.

        PHP Variables

        1.In PHP, the variable is a name or symbol which stands for value and used to store values such as characters, numeric, character string and memory addresses.

        2.Variables require a dollar sign ($) to be executed.

        3.It is possible to define a variable by simple assignment.

        4.We can redefine or undefine after it gets set.
