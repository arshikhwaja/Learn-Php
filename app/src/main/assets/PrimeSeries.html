Prime Series


A number which is only divisible by 1 and itself is called prime number.

Logic:
  • Take a number
  • Divide it by 2
  • If the remainder is 0, set flag 0 and break the loop
  • Print all the prime numbers

Program:
<?php
echo "Prime Series in first 20 numbers: <br> ";				
$num = 20;
for($i=2; $i<=$num; $i++)
{
$flag = 1;
for($n=2; $n<=$i; $n++)
{
 if($i%$n == 0)
 {
  $flag = 0;
  break;
 }
}
if($flag == 1)
{
 echo $n."<br>";
}
}
?>
