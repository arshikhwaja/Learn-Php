        Here are the rules for determine the "truth" of any value not already of a boolean type:

        1.If the value is a number,it is false exactly equal to 0 and true otherwise.

        2.If the value is string,it is false if the string is empty has 0 characters or if the string is "0",and is true otherwise.

        3.Values of type NULL are always false.

        4.If the value is an array,it is always false if it contains no other values,and it is true otherwise.For an object containing a value means having a member variable that has been       assigned a value.

        5.Valid resouces are true although some functions that return resources when they are successful will return FALSE when unsuccessful.

        6.Don't use doubles as booleans.
