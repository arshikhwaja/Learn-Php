Reverse Of a String


Logic:
  • Assign the string to a variable
  • Calculate the length of string
  • Declare a variable to hold reverse string
  • Run for loop
  • Concatenate string inside for loop
  • Display reverse string

Program:
<?php				
$string = "Jupiter";
$length = strlen($string);
for($i=($length - 1);$i >= 0;$i--)
{
 echo $string[$i];
}
?>

Output:

   retipuJ
