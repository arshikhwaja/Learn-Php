PHP Example

It is very to create a simple PHP example. To do so, create a file and write HTML tags + PHP code and save this file with .php extension. All PHP code goas between php tag. A syntex of PHP tag given below:

	<?php
	    //your code here
	?>

Let's see a simple PHP example wherevwe are writing some text using PHP echo command. File: hello.php

	⁢!DOCTYPE>

	 <body>

	     echo "Hello World!";
	   ?>	
	 </body>
	<html>
