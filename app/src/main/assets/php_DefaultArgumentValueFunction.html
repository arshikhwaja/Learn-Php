PHP Default Argument Value Function

Information can be passed to functions through arguments. An argument is just like a variable. Arguments are specified after the function name, inside the parentheses. You can add as many arguments as you want, just separate them with a comma.PHP also allows you to assign default values to the parameters, so when we call the function without arguments then it will take the assigned value of the particular parameter.

Syntax: 

	<?php
	
		function functionName(para1=10,para2,..paran) 
		{
			//code to be executed;
			
		}
		functionName();
		
	?>
