Largest of three numbers


Logic:
  • Take three numbers
  • Check whether the first number is greater than other two, if yes print first number is greatest
  • Else check whether the second number is greater than the third, if yes print second number is greatest
  • If both the above condition is false, print third number is greatest

Program:
<?php				
$a = 4;
$b = 16;
$c = 5;
if(($a>$b) && ($a>$c))
{
 echo $a." is greatest number";
}
else if($b>$c)
{
 echo $b." is greatest number";
}
else
{
 echo $c." is greatest number";
}
?>

Output:

  16 is greatest number
