PHP is_bool() function

By using this function, we can check the input variable is integer or not. This function was introduced in PHP 4.0.

    Syntex :

	<?php	
	bool is_int(mixed $var)
	?>

    Parameter:
     var_name: The variable being checked and it is compulsory.

    Returns:
     This function returns true if var_name is an integer, Otherwise false.

    Example 1 :

	<?php	
	$a=4;
	echo is_int($a);
	?>

    Output:


               1

    Example 2 :

	<?php	
	$a=5;
	is(is_int($a){
	echo $a." is an int!";
        }
	else{
		echo $a." is not an int!";
        }
	?>

    Output:


               5 is an int!
